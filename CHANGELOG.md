# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2025-01-28

### Fixed

- Updated Apploye UI selectors to match the new interface design
  - Badge selector: `.sc-hEUNDx.bnCgVq` → `.sc-jGONNV.cSDwqo`
  - Month header selector: `.sc-hQQMGV.hoIpyi` → `.sc-gzpZyF.eyiZkS`
- Separated all selectors into a dedicated constants file for easier maintenance
- Improved badge styling with better spacing and rounded corners

## [1.0.0] - 2024-XX-XX

### Added

- Real-time progress tracking badge on Apploye timesheet page
- Monthly progress indicator showing actual vs expected hours
- Holiday management with visual indicators on calendar
- Smart month-aware calculations
- Popup interface with detailed progress statistics
- Settings page for configuring daily target hours and holidays
- Chrome extension support
